### Introduction
A simplified scaffold refer to [vue-cli](https://github.com/vuejs/vue-cli), use [this template](https://github.com/2eron/vue-scaffold-template)

### Install
```
$ npm install -g vue-scaffold
```

### Usage
```
$ vs init project-name
$ cd project-name
$ npm install
$ npm run dev
```

### Build
```
$ npm run build
```
